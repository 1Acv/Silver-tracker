# Silver-tracker
Tracks the price of silver and junk silver

This is a simple script to track the price of silver per ounce and junk silver. Which for those that are unfamiliar with the term refers to US coinage that was made before 1965. This tracker goes further and includes many western countries that produced currency in silver from the 1900s-1970s. The script maps out the melt value of the silver inside the coins using weight calculates the USD price, with the goal of it being an on hand script for determining market value when buying and selling junk silver.
